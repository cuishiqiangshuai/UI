Git作业

学习目标：

  1. 掌握Git的基本操作，包括仓库初始化（git init），添加到暂存区（git add），提交（git commit），查看日志（git log），查看状态（git status），查看改变的内容（git diff），移除文件（git rm），重命名文件（git mv）
  2. 掌握Git的撤销操作，包括撤销工作区文件的修改（git checkout -- 和 git stash），把修改从暂存区撤回到工作区（git reset HEAD）
  3. 掌握如何打tag和查看tag（git tag）
  4. 知道Git的配置文件为~/.gitconfig，掌握如何添加忽略规则(.gitignore)
  5. 了解Git的分支原理，掌握如何创建(git br )、切换(git co )、删除(git br -d/-D )、合并分支（git merge ）
  6. 掌握Git的远端操作，包括克隆（git clone）、添加（git remote add）、推送（git push）、拉取（git pull）
